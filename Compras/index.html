Compras

Pedido de compra

Menu: Compras -> Pedido de compras

O Pedido de Compra é criado quando se tem a necessidade de comprar algo em que a aprovação seja por parte de operações, compras (exceto compra de ativos) e RH

Legenda (Cada status pedido)

Pedido de mercadoria

Menu: Compras -> Pedido de compras -> Novo -> Pedido de mercadoria

É utilizado para pedir tudo que não é para prestadores de serviço (reembolso para gerentes são considerados pagamento a prestadores), Exemplos em que é utilizado o pedido de mercadoria:

  • Mercadoria
  • Material de escritorio
  • Material de limpeza

Criar um pedido de compra:

Procurar fornecedor:

O nome do fornecedor pode ser apresentado tanto pela razão social quanto pelo nome fantasia. Se você estiver tendo dificuldades para localizar um fornecedor, experimente inserir o código F+CNPJ (apenas os números do CNPJ) como exemplo: F0603861000160.

Data de entrega:

Não é permitido definir a data de entrega para um dia anterior à data atual. Se o pedido já foi entregue, por favor, insira a data atual.

Itens do pedido

Produto (mercadoria):
O B1 Food mostra nessa seleção os itens de todas as nossas franquias, recomendamos que vocês lancem os itens no pedido com a lista de contagem como guia, pois todos os itens que tem em estoque, podem ser pedidos e não é incomum encontrarmos pedidos com itens que não fazem parte do estoque da loja

Não é mercadoria ?:
Atualmente não estamos controlando estoque dos itens que não são mercadoria, então em caso de material de escritorio e limpeza, podem utilizar os itens genéricos:
(D000307: MATERIAL DE LIMPEZA) / (D000308 - MATERIAL DE ESCRITORIO)

Quantidade e unidade de medida:
O que é importante mesmo é a quantidade que vai ser pedido para entrar no estoque, exemplo:

Quantidade: 2
Unidade de medida: KG

Tem o mesmo efeito que:

Quantidade: 1
Unidade de medida: PCT2KG

Preço: Colocar valor aproximado, referente a unidade de medida que foi informada

Utilização: Não precisa preencher

Adicionar mais itens: Clicar no botão verde com simbolo de +
Excluir item: Ao final das especificações do item, clicar no botão vermelho com simbolo de X

Observações:

Seguir esse modelo:
Localidade da loja + final cnpj + resumo do que é esta sendo pedido

Exemplos:
* Campolim 0150 - Mercadoria
* Campinas 1805 - Material de escritorio
* Porto alegre 2011 - Material de limpeza

Em caso de pedido da Euro, seguir esse modelo:
Localidade da loja + final cnpj + nome da rota
Exemplos:
* Campolim 0150 - Agosto 1
* Campinas 1805 - Setembro 2
* Porto alegre 2011 - Outubro 1

Pedido de serviço

Menu: Compras -> Pedido de compras -> Novo -> Pedido de serviço

É Utilizado para pagamento de prestadores de serviço e pedido de reembolso para gerentes

  • Prestadores onde não requer aprovação do Mauricio (Rotina)
  • Reembolsos de uber, combustiveis e afins

Procurar fornecedor:

O nome do fornecedor pode ser apresentado tanto pela razão social quanto pelo nome fantasia. Se você estiver tendo dificuldades para localizar um fornecedor, experimente inserir o código F+CNPJ (apenas os números do CNPJ) como exemplo: F0603861000160.

Caso de reembolso: Sempre que houver reembolso para gerentes, utilizar o fornecedor FParceiro

Buscar o item, colocar a quantidade e preço, utilização não precisa ser preenchido
Quando houver alguma duvida sobre qual produto colocar, entrar em contato com setor de compras

Observações:

Seguir esse modelo:
Localidade da loja + final cnpj + resumo do que é esta sendo pedido

Exemplos:
* Campolim 0150 - reembolso combustivel
* Campinas 1805 - troca de tomada nucleo
* Porto alegre 2011 - reembolso de uber

Solicitação de compra

Menu: Compras -> Solicitação de compra

A solicitação de compra é utilizado para solicitar a compra ou contratação de um prestador de serviço que requer autorização do Mauricio, quando é feita uma solicitação de compra, vai ser encaminhado ao Mauricio, com a aprovação dele no sistema, o setor de compras vai gerar um pedido para que seja possivel lançar a nota fiscal do prestador ou compra

Solicitação de compra mercadoria

Menu: Compras -> Solicitação de compra -> Novo -> Solicitação de Mercadoria

Utilizado para solicitar, uniformes, cadeiras, mesas, Tv's, microondas e afins

**Não requer que você selecione o fornecedor, pois essa selecão será feita pela matriz

Data de recebimento: Colocar data que o produto vai ser entregue, caso não saiba, coloque data atual

Preencher produto, quantidade, unidade de medida preço (estimado)

Caso tenha duvida sobre qual item colocar, solicitar a compras orientações

Centro de custo: Compras

Observações:

Seguir modelo:
Localidade - final do CNPJ - colocar um detalhes da proposta que já está em andamento, como forma de pagamento e afins (se houver)

Exemplo:
Galeão - 1988 - Compra de microondas, casas bahia, cartão de credito até 3x sem juros
Fortaleza - 1040 - Conserto da mesa quebrada, prestador oswaldo pede 50% de sinal e 50% quando terminar o serviço, pagamento via pix, boleto ou dinheiro
